# SmartMeet AI – Appointment Scheduling Platform

SmartMeet AI is an AI-powered appointment scheduling platform that allows users to easily book meetings through a calendar-based interface.

## Features

- Calendar-based meeting scheduling
- AI-powered meeting preparation assistant
- Double booking prevention
- Admin dashboard for managing appointments
- Public booking link for easy scheduling
- Backend database using Supabase

## Tech Stack

- Frontend: Mattr AI Platform
- Backend: Supabase (PostgreSQL)
- AI Features: AI scheduling suggestions and meeting preparation assistant

## Project Description

This project was built as part of the UPTIQ AI app development assignment. The goal was to create an AI-powered scheduling solution similar to Calendly that simplifies appointment booking and prevents scheduling conflicts.
